## 矩阵基础  
### 建立矩阵  
```MATLAB
A = [1,2,3;4,5,6]  
A = [0:1:10]
```
### 扩展矩阵  
```MATLAB
A = [A,A]
A = [A;A]
A(100,100) = x   %将下标(100,100)的元素置为x,未赋值的矩阵元素被赋值为0  
```
### 引用矩阵元素  
```
A =   
[ 1 2 3  
  4 5 6 ]  
```  
1. 下标引用  
A(1,2) = 2 ,按行列编号   
2. 序号引用  
A(3) = 2 , 按列编号
### 矩阵重排列  
```
A =   
[ 1 2 3  
  4 5 6 ]
```
1. reshape  
```
A = reshap(A,3,2)
A = 
[ 1 5
  4 3
  2 6 ]
```  
2. 冒号运算  
```
A(:)
```
堆叠成1列矩阵
